# knowledge-base/emergent/

> Unvalidated patterns. Things we've seen 1–2 times but haven't earned a place in `business/` or `methodology/` yet.

## Why this folder exists

We're a 2-rep team. Most signals look real on first encounter and don't hold up. Emergent is a holding pen — the pattern gets written down, dated, and tagged. If 3+ more deals back it, it graduates to `business/`. If it doesn't, it gets demoted (we keep the file for history).

## Graduation rules

| State | Criteria |
|---|---|
| **emergent** | 1–2 supporting observations |
| **validated** | 3+ supporting observations across distinct accounts |
| **canonical** | Referenced from `foundation/_synthesis.md` |
| **deprecated** | Pattern stops predicting; explicit demotion |

## Example files this folder will hold

- `board-comp-review-as-trigger.md` — saw it predict on one deal; need 2 more
- `cfo-pave-skepticism.md` — anecdotal pattern from podcasts; need more data
- `comp-band-refresh-after-funding.md` — funding → band refresh in 90 days?

## What does NOT go here

- Random hunches without an observation
- Patterns that contradict `foundation/_synthesis.md` *without* a clear contradicting observation
- Ideas (those go to `ideas/`)
