# data/

Reserved for sample input shapes that illustrate the lake layout described in [`../case-study/part2-data-foundation.md`](../case-study/part2-data-foundation.md) — e.g. example `trigger_events` rows, `account_context` JSON, `outcomes` schema.

Intentionally empty in this submission: no real Mento data has been seen, and synthetic samples would risk implying assumptions about HubSpot property names or Avoma schema that haven't been validated. The shapes are documented in prose inside Part 2 and Part 3.
